Deployment Explanation

This mobile application is deployed using GitHub Pages to provide a secure, accessible cloud-hosted landing page. The signed APK is available for direct download, allowing evaluators to install and test the app on their own devices. GitHub Pages was selected for its simplicity, reliability, and alignment with static deployment requirements. This approach ensures the app is accessible without requiring app store registration or additional infrastructure.
